# PDC-Project
TDS repository will include the work and progress of each contributor. The purpose of this TD Shooter is to help visualize how parallel and distributed processes can be divided across multiple threads and the performance changes under weak vs strong scaling. 

Rather than building a full on 2d game with shaders and shapes... we perfer to focus on the real time visuals of the concepts:

- Parallel Processing
- Load Balancing
- Strong vs Weak Scaling
- Performance(FPS/Workload size)
